Dragon机器人操作手册

一、基本控制命令
1. 机器人前进 - 控制机器人向前移动
2. 机器人后退 - 控制机器人向后移动
3. 机器人左转 - 控制机器人向左转向
4. 机器人右转 - 控制机器人向右转向
5. 机器人停止 - 立即停止机器人运动

二、关节控制命令
1. 抬起左手 - 左臂向上抬起90度
2. 抬起右手 - 右臂向上抬起90度
3. 放下左手 - 左臂放下到0度位置
4. 放下右手 - 右臂放下到0度位置

三、安全注意事项
1. 使用前请确保周围环境安全
2. 机器人运动时请保持安全距离
3. 遇到紧急情况立即说"机器人停止"
4. 定期检查机器人关节状态

四、故障排除
1. 如果语音识别不准确，请调节麦克风音量
2. 如果机器人无响应，请检查ROS连接状态
3. 如果音频播放异常，请检查音频设备配置
